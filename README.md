## Descripción del proyecto

En este archivo se realizó el proceso de ETL para dos tipos de datos:

- Texto
- Imágenes

A ambos conjuntos de datos se les aplicará un método de clasificación.

## Dataset

El dataset correspondiente a los **embeddings de texto** se encuentra disponible dentro de la carpeta del proyecto.

Sin embargo, en el caso de las **imágenes**, no fue posible incluirlas directamente en el repositorio debido a la gran cantidad de archivos y su tamaño.

Por esta razón, se proporciona un enlace a Google Drive donde se encuentra la misma información del proyecto, incluyendo las imágenes necesarias para ejecutar correctamente el proceso:

[Acceder al dataset de imágenes en Google Drive](https://drive.google.com/drive/folders/1CY_Vbs5yjN661opOCEICQK9NojWBkXHG?usp=sharing)
